## Code Architecture

### Overview
This is a React application with reusable UI components. The project focuses on accessible, testable components.

### Components
- **FeedbackButton** - A button component that opens a GitHub issue creation dialog. Props:
  - `repositoryUrl` (string, optional) - GitHub repository URL for feedback issues

### Key Patterns
- Components are functional and use hooks
- All user-facing components have comprehensive unit tests using React Testing Library
- Accessibility attributes (aria-labels, data-testid) are included for testability and usability

## Common Development Tasks

### Using the FeedbackButton Component

```jsx
import { FeedbackButton } from './FeedbackButton';

// In your component:
<FeedbackButton repositoryUrl="https://github.com/your-org/your-repo" />
```

The button automatically opens a GitHub issue creation dialog when clicked. Customize the repository URL to point to your feedback repository.
